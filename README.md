# Risk Analytics & Fraud Detection Dashboard

## Project Overview

This project demonstrates an end-to-end Risk Analytics & Fraud Detection solution using SQL Server, SSIS, Data Warehouse, and Power BI. It analyzes financial transaction data to identify fraud patterns, monitor risk indicators, and provide interactive dashboards for decision-making.

## Technology Stack

- SQL Server 2022
- SQL Server Integration Services (SSIS)
- Power BI
- DAX
- Power Query
